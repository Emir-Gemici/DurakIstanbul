# DurakIstanbul
İstanbul'un 39 ilçesindeki İETT hatlarını bilgi yarışmasına dönüştüren Android simülasyonu. "Rasgele" ve "Full" modlarıyla oynayın, %85 başarıya ulaşıp bölge uzmanlık sertifikalarını toplayın. Dinamik zorluktaki efsanevi İETT kullanıcılarına karşı yarışarak ilçe bazlı liderlik tablosunda zirveye yerleşin. Ulaşım bilginizi eğlenerek test edin!
